# 발표용 스크린샷 체크리스트

PPT 비교 슬라이드(10~11)에 **RecycleAI 실기기 캡처**가 필요합니다.  
아래 파일명으로 저장한 뒤 `python docs/presentation/build_presentation.py` 를 다시 실행하세요.

## 저장 위치

`docs/presentation/assets/`

| 파일명 | 촬영 화면 | 팁 |
|--------|-----------|-----|
| `recycleai_scan.png` | 카메라 + 초록 네온 박스 + 상단 지자체명 | 물건 1개만 비추기 |
| `recycleai_card.png` | 하단 결과 카드(픽토그램·배출 요일·팁) | 핀치 줌 1장 포함하면 더 좋음 |
| `recycleai_drag.png` | (선택) 주황 드래그 박스 + 「이 영역 분석」 | 차별점 강조용 |
| `recycleai_idle.png` | (선택) 대기 화면·사용 가이드 | 첫인상 슬라이드용 |

## 경쟁 앱 (이미 포함됨)

`competitor_*.png` — 대한민국 정책브리핑(2025.05) 공개 캡처.  
최신 UI와 다를 수 있으므로, 직접 「내 손안의 분리배출」을 설치해 캡처해 교체해도 됩니다.

## 캡처 방법 (Android)

1. 앱 실행 → 위치·카메라 허용  
2. 전원+볼륨다운 (또는 제스처) 스크린샷  
3. PC로 전송 후 위 경로에 복사  

캡처를 보내 주시면 PPT에 반영해 드릴 수 있습니다.
