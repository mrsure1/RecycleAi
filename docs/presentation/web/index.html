01 Problem

왜 만들었나

이사·출장·여행으로 '동네 규칙'이 바뀔 때

  • 😰35년 강남 생활 후 일산 이사한 60대 — 분리수거 경고 스티커를 두 번 받음
  • 🧤고무장갑·즉석밥 용기 등 '예전 동네 습관'이 오히려 실수를 만듦
  • 🔄지자체마다 분류가 반대인 경우도 있음 → 과태료·수거 거부
  • ⏰하루 6~12회 분리수거, 의심 건 1~3분씩 × 주당 10~15회
  • 📄기존 대안: PDF(글자 작음), 이웃질문(부담), 검색 앱(타 지역 기준)
